◆ LoadOverTime()

void cmurphy.fastpool.FastPool.LoadOverTime ( )

If pool is empty, instantiate a number of Prefabs equivalent to the initial quantity specified in the inspector, spread over multiple frames according to the increment count specified in the inspector. For instance, with an initial quantity of 100 and an increment count of 8, the pool will load 8 instances per frames, for a total of Ceil(100 / 8) = 13 frames.
This method spreads the load over multiple frames. To load everything in one frame, use Load() instead.
Will be automatically called if m_loadOnAwake is set to true and m_initialLoadSpread is set to true.
